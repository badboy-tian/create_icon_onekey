# create_icon

* 一键创建APP桌面图标，输入1024x1024的圆角图，输出android图片和ios图标。
* Create APP desktop icons with one click, input 1024x1024 rounded images, output android images and ios icons.   

* 支持的平台：Windows、macOS  
* Supported platforms: Windows, macOS
